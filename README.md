# DDanzi-ProHubanger

This python scripts are for crawling 'www.ddanzi.com/free' website's special articles.

Main Entry : pro_hubanger.py

Parameters
  - argv[1] : user id(eg: chicken_head)
  - argv[2] : password(eg: **************)
  - argv[3] : start page number(eg: 1)
  - argv[4] : end page number(eg: 1000)
  - argv[5] : past search period
